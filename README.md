# Kana App — S4 Projet Fil Rouge

Application web d'apprentissage des hiragana et katakana.

## Fonctionnalités
- Mode étude : visualiser les 67 caractères japonais
- Mode quiz : tester ses connaissances avec score et feedback
- Mode inverse : rōmaji → caractère
- Sélection des lignes à inclure dans le quiz
- Meilleur score persisté entre les sessions

## Stack
React · TypeScript · Vite · React Router · CSS Modules
